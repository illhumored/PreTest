# PreTest
KakaoPay PreTest Project

1. 프로젝트 소개
>- 사용자가 입력한 URL 을 짧게 변환(이하 단축) 후 출력합니다.
>- 단축 URL 입력 시, 원래 URL 로 리다이렉트(새창) 시켜줍니다.

2. 문제해결 전략 
>2-1. 단축 주소 변환
>>- 사용자가 입력한 URL을 기준으로 DB에 단축 URL 처리가 되어 있는지 확인합니다.
>>- 있을 경우, 해당 단축 URL 을 조회 후 화면에 출력합니다.
>>- 없을 경우, 가장 최근에 DB에 insert 된 ID 값을 가져와 +1 시킨 후 이에 대한 62진수 변환을 진행합니다. 이때, 변환된 62진수의 각 자리수의 index 값을 이용하여 임의로 정해놓은 62개의 배열(알파벳과 숫자)에서 값을 가져옵니다. 이렇게 62진수를 배열값의 index 값으로 맵핑하여 가져온 값을 DB에 insert 후 단축된 URL를 화면에 출력합니다.

>2-2. 단축 주소 리다이렉트
>>- 사용자가 입력한 URL을 기준으로 DB에 단축 URL 처리가 되어 있는지 확인합니다.
>>- 있을 경우, 그에 상응하는 원래 URL 정보를 가져와 화면에서 리다이렉트 합니다.
>>- 없을 경우, DB 내에 해당 단축주소가 없음을 사용자에게 알려줍니다.

3. 프로젝트 빌드 및 실행방법
>- Gradle 빌드 후 http://localhost 로 실행합니다. (포트는 80으로 설정되어 있습니다.)
>- AWS RDS 를 연동하였기 때문에 별도의 DB 셋팅은 필요하지 않습니다. (다만, 무료 인스턴스라 DB 조회에 시간이 다소 걸립니다.)
